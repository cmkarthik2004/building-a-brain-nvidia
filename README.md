# Building A Brain – NVIDIA AI Project

## Overview
**Building A Brain** is an AI project completed as part of the NVIDIA Learning Program. The project introduces the fundamental concepts of artificial intelligence and neural networks, demonstrating how AI models learn hidden patterns from data through training and inference.

## Objectives
- Understand the basics of neural networks.
- Learn how AI models identify hidden patterns in data.
- Explore the model training and inference process.
- Gain hands-on experience using Google Colab and NVIDIA learning resources.

## Technologies Used
- Python
- Google Colab
- Jupyter Notebook
- NVIDIA Learning Platform
